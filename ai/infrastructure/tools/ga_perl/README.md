# Perl GA Demo

A minimal genetic algorithm scaffold in Perl.

- Initializes a population of bitstring genomes
- Applies simple bit-flip mutation
- Prints initial and mutated populations

Environment variables:
- `GA_POP_SIZE` (default: 20)
- `GA_GENOME_LEN` (default: 16)
- `GA_MUT_RATE` (default: 0.05)

Run via VS Code task "Run Perl GA Demo" or directly:

perl ai/tools/ga_perl/ga.pl
